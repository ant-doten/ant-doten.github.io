# Self-Enrollment - Android OS

### Thank you for your participation in our Bring Your Own Device (BYOD) pilot program. Your participation and feedback in this pilot will provide valuable information to the Information Technology department. If you have any questions, concerns, or general feedback please email us at AndroidBYODPilot@xyz.com


Bring Your Own Device (BYOD) Pilot - Enrollment Guide
{: .fs-5 .fw-700 }

#### 0. Play Store

```
- Open 'Play Store'
```

<img src="/assets/images/android/android-mdm-00.png" alt="Play Store Icon" style="width:100px;height:100px;">

#### 1. Play Store - Hub

```
- Search 'Hub' 
- Select 'Intelligent Hub'
```

<img src="/assets/images/android/android-mdm-01.png" alt="Play Store - Search" style="width:250px;height:470px;">

#### 2. Play Store - Intelligent Hub Install

```
- Select 'Install' (This process may take a few minutes)
```

<img src="/assets/images/android/android-mdm-02.png" alt="Play Store - Intelligent Hub" style="width:250px;height:470px;">

#### 3. Play Store - Intelligent Hub Install Complete

```
- Select 'Open'
```

<img src="/assets/images/android/android-mdm-03.png" alt="Play Store - Intelligent Hub Open" style="width:250px;height:470px;">

#### 4. Intelligent Hub - Email Address

```
- Enter your email address (e.g., username@domain.com)
- Select 'Next'
```

<img src="/assets/images/android/android-mdm-04.png" alt="Hub Application - Email Address" style="width:250px;height:470px;">

#### 5. Intelligent Hub - Workspace ONE UEM

```
- Select 'Next'to being enrollment
```

<img src="/assets/images/android/android-mdm-05.png" alt="Hub Application - Workspace ONE UEM" style="width:250px;height:470px;">

#### 6. Intelligent Hub - User Information

```
- Enter 'Username' (e.g., domain\username)
- Enter 'Password'
- Please Note: Email Address Will Be Pre-Populated
- Select 'Next'
```

<img src="/assets/images/android/android-mdm-06.png" alt="Hub Application - User Information" style="width:250px;height:470px;">

#### 7. Intelligent Hub - Ownership

```
- Select 'Employee Owned'
- Select 'Next'
```

<img src="/assets/images/android/android-mdm-07.png" alt="Hub Application - Ownership" style="width:250px;height:470px;">

#### 8. Intelligent Hub - Enrollment

```
- Please Wait: Enrollment is in Progress (Please Note: This can take several minutes)
```

<img src="/assets/images/android/android-mdm-08.png" alt="Hub Application - Enrollment" style="width:250px;height:470px;">

#### 9. Intelligent Hub - Privacy

```
- Select 'I Understand'
```

<img src="/assets/images/android/android-mdm-09.png" alt="Hub Application - Privacy" style="width:250px;height:470px;">

#### 10. Intelligent Hub - Data Sharing

```
- Select 'I Agree' to share app experience information (Optional)
```

<img src="/assets/images/android/android-mdm-10.png" alt="Hub Application - Data Sharing" style="width:250px;height:470px;">

#### 11. Samsung Knox - Work Profile (Optional)

```
- Select 'Continue' to Setup Work Profile (Applicable to Samsung Android Devices)
```

<img src="/assets/images/android/android-mdm-11.png" alt="Samsung Knox - Work Profile" style="width:250px;height:470px;">

#### 12. Samsung Knox - Work Profile Creation 

```
- Creating Work Profile... (This may take several minutes)
```

<img src="/assets/images/android/android-mdm-12.png" alt="Samsung Knox - Work Profile Creation" style="width:250px;height:470px;">


#### 13. Hub Application - Enrollment Complete

```
- Your device has been successfully enrolled! 
- Thank you for your participation in our BYOD pilot program!
```

<img src="/assets/images/android/android-mdm-13.png" alt="Hub Application - Enrollment Complete" style="width:250px;height:470px;">
