# Self-Enrollment - Apple iOS

### Thank you for your participation in our Mobile Device Management pilot. Your participation and feedback in this pilot will provide valuable information to the Information Technology department. If you have any questions, concerns, or general feedback please email us at AppleBYODPilot@xyz.com

#### 1. Navigate to Apple App Store

```
- Select 'Search'
```

<img src="/assets/images/apple/apple-mdm-01.jpg" alt="Apple App Store" style="width:275px;height:470px;">

#### 2. Search for 'Intelligent Hub' in Apple App Store

```
- Select 'Get' (Please Note: This May Require You to Enter Your Apple ID Password.)
```

<img src="/assets/images/apple/apple-mdm-02.jpg" alt="Hub App Store" style="width:275px;height:470px;">

#### 3. Hub Application

```
- Locate 'Hub' Application on iOS Home Screen.
```

<img src="/assets/images/apple/apple-mdm-03.jpg" alt="Hub Application" style="width:100px;height:120px;">

#### 4. Hub Application - Email

```
- Enter Your Email Address (E.g., username@domain.com)
- Select 'Next'
```

<img src="/assets/images/apple/apple-mdm-04.jpg" alt="Hub Email Information" style="width:275px;height:470px;">

#### 5. Hub Application - User Account

```
- Enter Your User Account (E.g., Domain\Username)
- Enter User Credentials (Password)
- Select 'Next'
```

<img src="/assets/images/apple/apple-mdm-05.png" alt="Hub User Information" style="width:275px;height:470px;">

#### 6. Hub Application - Employee Owned

```
- Select 'Employee Owned' for BYOD Enrollment
- Select 'Next'
```

<img src="/assets/images/apple/apple-mdm-06.png" alt="Hub BYOD Enrollment" style="width:275px;height:470px;">

#### 7. Hub Application - Privacy

```
- Select 'We don't collect' to understand what information isn't collected by MDM
- Select 'We may collect' to understand what MDM may collect
- Select 'Next'
```

<img src="/assets/images/apple/apple-mdm-07.png" alt="Hub - Privacy" style="width:480px;height:560px;">

#### 8. Hub Application - Notifications

```
- Select 'Allow' on Hub Notifications Prompt
```

<img src="/assets/images/apple/apple-mdm-08.png" alt="Hub - Notifications" style="width:275px;height:470px;">

#### 9. Configuration Profile

```
- Select 'Allow' on Configuration Profile (Please Note: This Opens Automatically in Web Browser)
```

<img src="/assets/images/apple/apple-mdm-09.png" alt="Configuration Profile" style="width:275px;height:470px;">

#### 10. Configuration Profile - Downloaded

```
- Select 'Close'
```

<img src="/assets/images/apple/apple-mdm-10.png" alt="Configuration Profile" style="width:275px;height:470px;">
